# Luxe & Little Treasures Business Manager

An iPhone-first PWA for managing inventory, sales, customers, expenses, reports, and receipts for Luxe & Little Treasures.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

Vercel can deploy this as a Vite app.

- Build command: `npm run build`
- Output directory: `dist`

## Data

The app stores business records locally in IndexedDB on the device. Use **More → Backup & Security → Export** regularly.
